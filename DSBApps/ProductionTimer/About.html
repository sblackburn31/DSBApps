Production Timer Emulator

What is This?

This is an emulation of an actual factory production timer screen that is part of a larger and much more sophisticated system. This has been presented to show off my programming abilities in creating a Web Application and to show that I can do Full stack development.

How to Use?

To use the app, select the # EMPLOYEES and a WORK CELL,
then enter a number in the PRODUCTION # text box.
The only option from there is to either Clear, Pause or Start.

Once you have started, the options are to Pause or End.

Once you have paused you can unpause, select another pause or end.

Once the End is selected the Productoin ends and the total time is displayed.

How was it Created?

Technologies used in this application are, on the front-end, JavaScript, HTML, CSS, AJAX and Bootstrap. On the back-end, this application makes REST calls to an ASP.Net MVC Web API that uses C# and Entity Framework to make calls to a SQL Server database that primarily uses Stored Procedures to retrieve and update data.
